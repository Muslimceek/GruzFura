# Груз ↔ Фура MVP

Платформа для поиска грузов и транспорта.

## Локальный запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

## Деплой на Vercel

1. Создайте репозиторий на GitHub и загрузите туда этот код.
2. Зайдите на [Vercel](https://vercel.com).
3. Нажмите "Add New..." -> "Project".
4. Импортируйте ваш репозиторий с GitHub.
5. Vercel автоматически определит настройки (Framework Preset: Vite).
6. Нажмите **Deploy**.

## База данных

Проект использует Firebase. Убедитесь, что ваш домен (на Vercel) добавлен в список авторизованных доменов в консоли Firebase Authentication (Settings -> Authorized Domains).
